Augmenting Vietnam map¶

Information is usually incomplete
The base kml map can be found online but information is usually incomplete and requires augmenting.

geopandas: kml as dataframes
geopandasallow kml information to be processed as dataframes, each column containing a class of information.

Vietnam administrative hierarchy At the lowest level, there are administrative units. Several of these combine into a province. Having such levels allows information, e.g. demographics, to be analyzed from different perspectives.

🛠️ Steps¶

  1. Load base KML into a GeoDataFrame
  2. Export admin unit names for manual labeling
  3. Merge labeled province data back into GeoDataFrame
  4. Export final enriched data kml
